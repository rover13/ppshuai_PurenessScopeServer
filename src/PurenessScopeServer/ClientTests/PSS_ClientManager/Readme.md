##此工具设计目的
>用于后台管理PSS的相关运行指标并提供对PSS服务器控制功能。  
##功能列表
* 管理服务器已经加载的模块信息，并且可以卸载或加载拔指定的插件。  
* 客户端连接管理，可以看到当前服务器的流量，以及CPU和内存的占用量。 
* 主模块运行信息，你可以在这个选项卡里面查询指定的命令调用次数和时间消耗。  
* 可以查看和关闭，当前服务器上已经连接上的客户端连接，并查看指定客户端链接的到达数据包数，发送包数，接收字节，发送字节等信息。  
* PSS服务器工作线程状态，可以显示当前工作线程的数据处理情况，以及对工作线程的监控报警。
* 可以查看当前服务器的服务器间连接信息，并管理发起重练等信息。并可以在这个界面上动态的添加新的PSS监听入口而不用重启服务器本身。
* IP封禁，可以在这个界面下封禁指定来源的客户端IP，支持永久封禁和时段封禁。
* IP追踪，主要提供IP的追踪信息，可以跟踪指定的IP时段内的连接行为。
* 工作线程AI模块，工作线程可以根据数据处理的快慢动态调整接口的服务状态。
